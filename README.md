# JU-qPCR-Analysis
qPCR Analysis pipeline for In Course Assessment (ICA) in New Targets and Drugs in Cancer Therapy Module.

## How to Use
- Download R/R studios
- Install packages used using install.packages("[package name]")

## Installed Packages Used are
- ggpattern_1.2.1
- ggrepel_0.9.6
- broom_1.0.7
- tidyr_1.3.1
- ggpubr_0.6.2
- readr_2.1.5
- ggplot2_4.0.0
- dplyr_1.1.4
